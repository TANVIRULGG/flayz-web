Get Started
Running in
Under 5 Minutes
01
Add Flayz
Click "Add to Discord" and authorize Flayz for your server. Done in 30 seconds.
02
Install & Configure
If self-hosting, install requirements and set up your .env file with your bot token and MongoDB URI.
